Place your .txt documents under the `data/` directory. Example:
data/manual1.txt
data/specs/spec_b.txt

Then run:
python ingest.py --data_dir data/ --out_raw chunks.jsonl
python index.py --chunks chunks.jsonl --index_path faiss_index.idx --meta_path docs_meta.pkl
uvicorn serve:app --host 0.0.0.0 --port 8000
streamlit run app.py
